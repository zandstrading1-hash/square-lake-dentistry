# Square Lake Family Dentistry — Website

A fast, responsive, **multi‑page** marketing site for **Square Lake Family Dentistry**, a
family‑owned dental practice in Troy, MI (owner: **Dr. Saif Hanna**). Built as plain
HTML/CSS/JS — no runtime dependencies — so it deploys anywhere static, including GitHub Pages.

Booking is **phone‑first**: every primary call‑to‑action is “Call to book” → `tel:(248) 558‑2785`.
There is no online appointment form; the Contact page has an optional “Send us a message”
form for non‑urgent questions only.

## Editing the site

You can edit any `.html` file directly — they're plain, readable HTML.

For convenience, the inner pages are also **generated** from small Python files so the
shared header/footer stay in sync across all ~20 pages. This is optional — if you prefer,
ignore the build script and just edit the HTML.

```
buildlib.py            Shared <head>, header, footer, CTA helpers, page renderer
content_core.py        About, Services hub, New Patients, Insurance, Reviews, FAQ, Contact
content_services.py    The 8 individual service pages
content_blog.py        Blog index + posts
build.py               Runs everything → writes the .html files + sitemap + robots
patch_index.py         One-off script used to align the hand-authored homepage
```

To regenerate the inner pages after editing content:

```bash
python3 build.py
```

> **Note:** `index.html` is hand‑authored (it has a custom hero/visual) and is **not**
> regenerated by `build.py`. Edit it directly.

## Get it live (GitHub Pages)

The repo is already a git repository with a commit. To publish:

**Option A — Claude Code (recommended).** Open Claude Code inside this folder and ask it to
create a public GitHub repo, push, and enable Pages. (The GitHub connector in the chat app
powers attaching/reading repos; the actual push/commit happens in Claude Code.)

**Option B — Manual.**
```bash
git remote add origin https://github.com/<your-username>/square-lake-dentistry.git
git branch -M main
git push -u origin main
```
Then on GitHub: **Settings → Pages → Source: `main` / root → Save.**
Your site appears at `https://<your-username>.github.io/square-lake-dentistry/` in a minute or two.

**Option C — Netlify Drop.** Drag this whole folder onto <https://app.netlify.com/drop>
for an instant live URL (no account needed to preview).

When you move to the real domain (squarelakefamilydentistry.com), the canonical/OG tags
already point there.
